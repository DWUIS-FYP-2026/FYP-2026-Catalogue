Supervisor project register

Update student project records through a structured workspace. When you save a record, the workspace commits the updated register file directly to the authorised GitHub repository. No separate database service is used.

Supervisor access

Use a GitHub account that has permission to update this repository. The workspace does not store a password or access token. Do not publish student passwords, private keys, protected reports, grades, or confidential material.

GitHub repository connection

Connect before editing records

This is an editing interface for the project register—not a text editor. It loads the current student data from assets/projects.js, lets you edit a selected record through the form, and commits the update to GitHub.

Each supervisor uses their own GitHub fine-grained personal access token with access limited to this repository. The token is held in this browser tab only and is discarded when you disconnect or close the tab.
