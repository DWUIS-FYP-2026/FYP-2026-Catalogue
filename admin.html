This website package contains the sign-in and editing interface, but Firebase has not yet been configured. Complete the setup guide, then return here to sign in.

  1. Create a Firebase project and register this website as a web app.
  2. Enable Email/Password sign-in and create the supervisor accounts.
  3. Set the Firestore rules supplied with this project.
  4. Copy the Firebase configuration into assets/firebase-config.js.
  5. Create an active supervisor record, sign in, then seed the supplied 25 project records.
