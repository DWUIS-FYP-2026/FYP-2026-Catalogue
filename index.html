Supervisor workspace

Same register, controlled updates

Use this workspace to create, review, update, or remove project records without opening or editing the project data file. Each confirmed change is committed to the authorised GitHub repository and appears in the public register after GitHub Pages publishes it.

Supervisor access

Use only a GitHub account with permission to update this repository. Do not add passwords, private keys, confidential reports, grades, or private student information.

GitHub repository connection

Connect before editing records

This is an editing interface for the project register—not a text editor. It loads the current student data from assets/projects.js, lets you edit a selected record through the form, and commits the update to GitHub.

Each supervisor uses their own GitHub fine-grained personal access token with access limited to this repository. The token is held in this browser tab only and is discarded when you disconnect or close the tab.

Read the supervisor connection guide

Create a fine-grained token for this repository only, with Contents: Read and write. Do not paste a token into any public document, email, or source file.
